# So, how do you fill a glass of water?

This repository contains a simple simulation of filling a glass of water using a tap. The simulation is implemented in C and demonstrates the use of the [CppModel](https://cppmodel.com) library for creating and running simulations.

## Getting Started

To get started with the simulation, sign in to your CppModel account and download the CppModel library for the platform you are using. Extract the library files into the `dependencies` folder of this repository. Using CMake, you can build the project and run the simulations.

## Blog Post
For a detailed explanation of the simulation and its implementation, please refer to the [blog post](https://cppmodel.com/blog/glass-of-water) on the CppModel website. The blog post provides insights into the design choices, the simulation logic, and how to extend the simulation further.
